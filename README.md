# PCA_and_Naive_bayes_implementation_from_scratch
there are 2 variations of naive bayes: 
1. categorial
2. numerical (using gaussian distribution)
I tried both on 2 different datasets, one of them is fully categorial and the other is fully numerical
and the results were:
1. categorail naive bayes with car_evaluation dataset: accuracy: 0.8323699421965318
2. numerical naive bayes with student placement dataset: accuracy: 0.6934

for PCA, it was also tested on 3 different datasets
